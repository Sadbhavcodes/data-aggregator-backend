## ЁЯза 3. `app.ts` тАФ The Nervous System

### ЁЯЪА Entry Point of the Application

```ts
import express, { Request, Response } from "express";

const app = express();
const port = process.env.PORT || 3000;

app.get('/', (req: Request, res: Response) => {
  res.send('Server Running');
});

app.listen(port, () => {
  console.log(`Server running on ${port}`);
});
```

---

### ЁЯФН Key Concepts

#### `app.get(path, callback)`

* Registers a route
* Waits for incoming requests

#### `req (Request)`

* Incoming data
* Represents **client input**

#### `res (Response)`

* ServerтАЩs reply
* Used to send data back

#### `app.listen(port)`

* Starts the server
* Opens port `3000`

| Part | Meaning     |
